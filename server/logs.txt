    /**
     * Merge two sorted subarrays into one sorted subarray.
     *
     * @param arr   the array to merge
     * @param left  the start index of the first subarray
     * @param mid   the end index of the first subarray and the start index of the second subarray
     * @param right the end index of the second subarray
     */
    public static void merge(int[] arr, int left, int mid, int right) {
        // Create temporary arrays to store the subarrays
        int[] leftArr = new int[mid - left + 1];
        int[] rightArr = new int[right - mid];

        System.arraycopy(arr, left, leftArr, 0, mid - left + 1);
        System.arraycopy(arr, mid + 1, rightArr, 0, right - mid);

        // Initialize indices for the temporary arrays
        int i = 0, j = 0;

        // Merge smaller elements first
        int k = left;
        while (i < leftArr.length && j < rightArr.length) {
            if (leftArr[i] <= rightArr[j]) {
                arr[k++] = leftArr[i++];
            } else {
                arr[k++] = rightArr[j++];
            }
        }

        // Copy remaining elements from the temporary arrays
        while (i < leftArr.length) {
            arr[k++] = leftArr[i++];
        }
        while (j < rightArr.length) {
            arr[k++] = rightArr[j++];
        }
    }

    /**
     * Sort an array using Merge Sort.
     *
     * @param arr   the array to sort
     * @param left  the start index of the subarray to sort
     * @param right the end index of the subarray to sort
     */
    public static void mergeSort(int[] arr, int left, int right) {
        if (left < right) {
            // Find the middle point and divide it into two halves
            int mid = (left + right) / 2;

            // Recursively call mergeSort on both halves
            mergeSort(arr, left, mid);
            mergeSort(arr, mid + 1, right);

            // Merge sorted subarrays
            merge(arr, left, mid, right);
        }
    }

**Why QuickSort?**

QuickSort has an average time complexity of O(n log n), making it suitable for large datasets. It also has an in-place sorting capability, meaning it doesn't require additional storage space.

**Algorithm Steps:**

1. **Choose a pivot**: Select a value from the array to be used as a reference point (pivot).
2. **Partition**: Arrange the elements into two groups:
	* Elements less than the pivot
	* Elements greater than the pivot
3. **Recursively sort subarrays**: Apply QuickSort to both the left and right partitions.

**Implementation in Python:**

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    less_than_pivot = [x for x in arr[1:] if x <= pivot]
    greater_than_pivot = [x for x in arr[1:] if x > pivot]
    return quicksort(less_than_pivot) + [pivot] + quicksort(greater_than_pivot)
```

**Example usage:**

```python
data = [5, 2, 8, 3, 1, 4, 6]
sorted_data = quicksort(data)
print(sorted_data)  # [1, 2, 3, 4, 5, 6, 8]
```

**Time complexity:**

* Best case: O(n log n)
* Average case: O(n log n)
* Worst case: O(n^2)

**Note:** In the worst-case scenario (when the pivot is always the smallest or largest element), QuickSort can degrade to O(n^2) performance. However, this is rare in practice and can be mitigated by using a better pivot selection method.
